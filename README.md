# Будущий ты — Telegram-бот

ИИ-агент публикует мотивирующие фразы в Telegram-канал **«Будущий ты»** **каждые 2 часа** (00:00, 02:00, 04:00 … по Москве). Сообщения пишутся от лица человека из будущего, у которого всё получилось.

---

## Что нужно заранее

1. **Telegram-бот** — создайте через [@BotFather](https://t.me/BotFather), получите токен.
2. **Канал** — создайте канал и добавьте бота **администратором** с правом публикации.
3. **Ключ DeepSeek** — [platform.deepseek.com](https://platform.deepseek.com/)

---

## Деплой на Railway (рекомендуется, 24/7)

Подробная инструкция: **[RAILWAY.md](RAILWAY.md)**

Кратко: GitHub → Railway → Variables → бот работает без включённого ноутбука.

## Локальный запуск (Windows)

### 1. Установите Python 3.10+

```powershell
py --version
```

### 2. Установите зависимости

```powershell
cd C:\Users\User\Desktop\futureyou
py -m pip install -r requirements.txt
```

### 3. Настройте `.env`

```powershell
copy .env.example .env
notepad .env
```

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_TOKEN` | Токен от @BotFather |
| `CHAT_ID` | `@username_канала` или `-100...` |
| `DEEPSEEK_API_KEY` | Ключ DeepSeek |
| `POST_INTERVAL_HOURS` | `2` — публикация каждые 2 часа |
| `TZ` | `Europe/Moscow` |
| `PROXY` | Прокси VPN, если Telegram заблокирован |

### 4. Запуск

```powershell
py bot.py
```

Тестовая публикация:

```powershell
py -c "from bot import publish_post; publish_post()"
```

## Как это работает

```
┌─────────────┐     ┌──────────────┐     ┌─────────────────┐
│  Каждые 2 ч │────▶│   DeepSeek   │────▶│  Telegram API   │
│  00:00...   │     │     ИИ       │     │  (канал)        │
└─────────────┘     └──────┬───────┘     └─────────────────┘
                           │ ошибка?
                           ▼
                    ┌──────────────┐
                    │ 10 запасных  │
                    │    фраз      │
                    └──────────────┘
```

- **12 постов в сутки** — каждые 2 часа по часовому поясу `TZ`
- **ИИ** — случайный промпт из трёх вариантов
- **Фильтр** — отсекает слова: лень, тупой, срочно, обязан
- **Ошибки** — до 3 повторных попыток

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| Telegram не открывается | Включите VPN, укажите `PROXY` в `.env` |
| Посты не в московском времени | `TZ=Europe/Moscow` |
| ИИ не отвечает | Проверьте баланс DeepSeek; сработают запасные фразы |
| Два поста сразу | Бот запущен и на ноуте, и на Railway — оставьте один |
